The "Camper Rent" Web Site

This is the website where you can search for a camper for your adventure! Choose
a camper and start your journey! Here you may find wide range of campers
filtering your search by camper`s type or features. You can easy book a camper
right now!

Stack of used technologies :

- Vite
- React, Redux
- Axios, Clsx, Formik, NanoId other libraries and frameworks

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and
some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md)
  uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc)
  uses [SWC](https://swc.rs/) for Fast Refresh
